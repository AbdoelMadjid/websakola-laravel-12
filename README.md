## Installation

```console
git clone https://github.com/AbdoelMadjid/websakola-unify
```

```console
cd websakola-unify
```

```console
composer install
```

```console
composer dump-autoload
```

```console
cp .env.example .env
```

```console
php artisan key:generate
```

```html
DB_CONNECTION=mysql DB_HOST=127.0.0.1 DB_PORT=3306 DB_DATABASE=??
DB_USERNAME=root DB_PASSWORD=
```

```console
php artisan migrate
```

```console
php artisan db:seed
```
